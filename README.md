# narexil-desktop

Waybar configuration for Hyprland.

## Requirements

- waybar
- hyprpaper
- Fonts: Rubik, Material Symbols Rounded

## Setup

```bash
git clone https://github.com/Esperadoce/narexil-desktop ~/.config/waybar
```

Or symlink from `~/Source/narexil-desktop` if developing.

## Screens

- `HDMI-A-1` — MSI OLED ultrawide — auto-hide bar, black wallpaper
- `DP-1` / `HDMI-A-2` — Dell 1440p — persistent bar

See `docs/design.md` for full architecture.
